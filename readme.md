# 使用说明

适用于`scholar.google.com`

先执行`easyscholar`插件判断哪些论文是`CCF-A`；

再执行本插件统计当前作者论文信息；



# 配置页

- **Show More Wait**: 每次加载`Show more`按钮后的等待时间，默认500ms;

- **After Expand Wait**: 全部论文加载完成后的等待时间，默认2000ms;
- **Minimal Display**: 是否精简显示结果;
- **Enable Debugging**: 是否再控制窗口输出调试信息;
